# **App Name**: RALFeedback

## Core Features:

- Restaurant Selection: Display of fast food restaurant options in a clear 3-column grid layout with clickable cards.
- Sentiment Indexing: AI-driven analysis to determine and display a "Sentiment Index" highlighting the top-rated and the restaurant most needing improvement.
- Feedback Display: Tabbed interface to separate and display customer reviews from employee feedback.
- Fawud System: Enable users to upvote feedback with the 'Fawud' system, allowing multiple 'fawuds' per comment.
- AI Feedback Scoring: Use a tool for AI sentiment scoring to analyze feedback and determine its quality and source (customer vs. employee).
- AI Summarization: Summarize sentiment, generate narrative summaries, and identify key positive/negative points using AI, focusing on service, staff, and employee feedback on management.
- Email Signup Popup: A one-time popup dialog that appears after a short delay for first-time visitors, prompting them to subscribe for real meal deals and discounts and a chance for a $20K per month meal card.

## Style Guidelines:

- Primary color: Vibrant Orange (#FF7B54), reminiscent of spice and energy.
- Background color: Light Tan (#FAF0E6), a desaturated near-tone of the primary color to create a light scheme with warmth and authenticity.
- Accent color: Coral (#FF8E72) to complement the primary color, drawing focus to key interactive elements such as the 'Summarize' button or prominent sections within the layout.
- Headline font: 'Belleza' sans-serif for headlines; body font: 'Alegreya' serif for readability in feedback sections.
- Subtle 'pulse' animation on the header logo to attract user attention without being intrusive.
- Cards used extensively to organize content, providing a tactile and modular feel in line with modern web design principles.
- Custom-designed icons that reflect local culture for main UI features such as Sentiment Index cards.
